Sobre o Projeto

Este projeto (“Evolução dos processadores”) foi realizado na disciplina de Introdução à Engenharia Informática, no curso de Engenharia de Computadores e Informática, na Universidade de Aveiro, tendo sido criado no dia 3 de janeiro de 2023. Cada um dos autores (“António Caetano e Leonardo Luís) teve uma contribuição de 50% neste projeto final, trabalhando juntamente tanto na elaboração do Índice, como no resumo dos diferentes conceitos em cada capítulo, a indexação de imagens e o posterior código sintetizado para cada uma das partes de forma a elaborar todo este site.
